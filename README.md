bad-bode
========

Bad Bode is a platform game made with Unity - Game Engine for my Digital Game Development Postgraduate Course at PUCRS.

It was developed together with <a href="http://www.linkedin.com/pub/felipe-peruzzo-milkewicz/72/7b2/189">Felipe Peruzzo Milkewicz</a>, <a href="http://www.linkedin.com/in/felipegalvao">Felipe Irigon Galvão</a> and <a href="http://www.linkedin.com/pub/gabriel-carvalho/65/556/174">Gabriel Carvalho</a>.
